Información importante para tu viaje

Las restricciones de viaje cambian con regularidad. Aquí podrás encontrar una lista de requisitos antes de viajar.

  • Asegúrate de tener toda la documentación Covid-19 necesaria para entrar a tu país de destino, ya que dependiendo de este, los requisitos pueden cambiar.
  • Sin esta documentación esencial, es posible que no puedas viajar.
  • La documentación necesaria de Covid-19 puede incluir, pero no está limitada a:
    1. Formulario de localización de pasajeros
    2. Certificado digital Covid
    3. Prueba de test antígeno/PCR negativa

Si desea conocer a fondo las restricciones en el país de destino, visite su página oficial. Algunas de ellas son:

  • Francia
  • Irlanda
  • Italia
  • España
  • Reino Unido
